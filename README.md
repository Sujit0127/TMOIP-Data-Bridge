## Overview

This project is a desktop-based application designed to receive, process, and transmit data in real time.

### Features
- Supports **Throughput Mode** and **Packet Mode** data reception
- Real-time data processing and frame validation
- Ethernet transmission using **UDP protocol**
- Implements **TMoIP (Time Multiplexed over IP)** for formatted output
- User-friendly GUI for configuration, monitoring, and logging

### Functionality
The application captures incoming data streams, processes them into structured frames, and transmits the formatted output over Ethernet using UDP. It provides visualization tools, logging, and configuration options for IP settings and synchronization parameters.

### Technologies Used
- Desktop GUI framework (e.g., PyQt / Tkinter / .NET / etc.)
- Networking: UDP sockets
- Protocol: TMoIP
